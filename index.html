HTML

  • HyperText Markup Language
  • É a camada principal de um site
  • Relaciona com todas as outras camadas
  • Camada do conteúdo em si, aqui estão todas as informações que vão ser exibidas para o usuário
  • O HTML é feito de um conjunto de etiquetas de conteúdo que marcam qual parte do texto é o que.
        <a href="http://www.google.com" id="link" class="primario">
          Fonte: wikipedia
        </a>

O atributo id define o nome único para essa etiqueta, enquanto o class define o comportamento que ele vai ter no nosso site.

HTML

O HTML é dividido em duas partes principais, a cabeça (<head>) e o corpo (<body>)

        <!DOCTYPE html>

        <head>

        </head>

        <body>

        </body>
        </html>

Títulos

HTML

Com o parágrafo temos uma exibição padrão de textos, precisamos agora definir os titulos da nossa página.

Eles são h1, h2, h3, h4, h5 e h6, com o h1 sendo o mais importante e o h6 o menos.

        <body>
          <h1>Olá mundo!</h1>
          <h2>Vestibulum eu odio</h2>
          <p>Lorem <strong>ipsum</strong> dolor <em>sit</em> amet!</p>
        </body>

O h1 deve existir apenas uma vez na página para definir qual o título mais importante. Os outros podem se repetir.

Todos os títulos devem seguir uma hierarquia, sendo que um h3 tem que ser um sub-titulo de um h2.

CSS

Cascading Styles Sheets

É como nós estilizamos o conteúdo de um documento HTML, afinal a internet não precisa ser somente branco e preto.

Sua estrutura é diferente da do HTML, no CSS nós vamos definir a cor, tipo de fonte e toda a apresentação que a página vai ter.

Aplicando o CSS

CSS

Para aplicar o css nós podemos defini-lo direto na etiqueta:

        <p style="color: red">texto</p>

Não é recomendado, pois todo os estilos estão direto no conteúdo do site, é preferivel ter uma definição de css.

        <head>
          <style>
            p {
                color: red;
            }
          </style>
        </head>

Aplicando o CSS

CSS

O metodo mais usado para aplicar folhas de estilo é importar um arquivo externo

        <head>
          <title>Cidades mais populosas do mundo</title>
          <link rel="stylesheet" media="all" href="estilo.css">
        </head>

Desta maneira nós conseguimos colocar em apenas um arquivo todo o css que vai ser usado em todo o site, sem precisar ficar duplicando conteúdo em várias páginas.

Seletores, propriedades e valores

CSS

Um seletor é como do CSS nós identificamos um elemento do HTML. Ele pode utiizar o nome do elemento, seu id ou sua classe.

Para cada seletor existem propriedades dentro de chaves que vão ser expressos por palavras simples, como color, font-weight ou border.

Um valor é dado para cada propriedada definindo como ela vai se comportar.

      body {
        font-size: 14px;
        color: navy;
      }

Cores

CSS

O css tem uma paleta de cores com 16 777 216 opções, podendo ser definidas por nome, hexa ou rgb

  • red
  • rgb(255,0,0)
  • rgb(100%,0%,0%)
  • #ff0000
  • #f00
      h1 {

Texto

CSS

Com estas propriedades é possivel alterar a forma e o tamanho do texto.

font-family

A fonte que vai ser usada, como Arial, Helvetica ou Comic Sans

font-size

O tamanho que o texto vai ser exibido

font-weight

O peso que a fonte vai ser exibida. Essa é uma propriedade mais nova, não sendo aceita por navegadores antigos.

Texto

CSS

font-style

O estilo da fonte, se está em italico ou não

text-decoration

"Decora" a fonte com uma linha que pode

  • text-decoration: underline sublinha o texto.
  • text-decoration: overline coloca uma linha sobre o texto.
  • text-decoration: line-through cola uma linha no texto, ricando o conteúdo.

Texto

CSS

text-transform

Altera a capitalização do texto

  • text-transform: capitalize faz com que a primeira letra esteja maiúscula.
  • text-transform: uppercase faz com que a todas letra estejam maiúsculas.
  • text-transform: lowercase faz com que a todas letra estejam minúsculas.

Espaçamento de Texto

CSS

Temos algumas propriedades que são especificas para trabalhar com o espaçamento de texto.

letter-spacing e word-spacing são para definir o espaçamento entre letras ou palavra.

line-height define o tamanho que a linha vai ter, sem alterar o tamanho que a fonte vai ter.

text-align troca o alinhamento padrão do texto, podendo ser para a esquerda, direita ou centralizado.

text-indent trata da identação da primeira linha de um parágrafo.
